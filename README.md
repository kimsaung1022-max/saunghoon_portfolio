# 김상훈의 학술 포트폴리오 웹사이트

##  프로젝트 개요
이 프로젝트는 웹프로그래밍 수업 중간고사 대체 과제로,  
학업 및 자기소개를 정리한 개인 포트폴리오 웹사이트입니다.  
간단한 HTML, CSS, JavaScript를 사용하여 제작하였습니다.

## 사용 기술
- HTML: 웹페이지 구조 설계
- CSS: 디자인과 레이아웃
- JavaScript: 메뉴 클릭 및 간단한 기능 구현

##  주요 내용
- 자기소개 페이지
